== Changelog ==

= 1.4.2 =
* Баг в привязке импортируемых товаров к категориям
* Исправлены ошибки в импорте товаров, возникавшие про определенных
* условиях

= 1.4.1 =
* Исправлены ошибки с импортом картинок, возникающие при определенных условиях
* Исправление по верстке
* Адаптация к использованию с темой GdeSlon Affiliate Shop Theme Begi

= 1.4.0 =
* Изменена работы с картинками - иллюстрациями товаров. Они (опционально) выкачиваются на сервер, на котором расположен плагин. Картинки стали обычными WP-картинками, добавлены в медиа-библиотеку.
* Исправлена ошибка выкачки файлов, связанная с CURLOPT_FOLLOWLOCATION
* Категории без товаров не показываются в виджете

= 1.3.3 =
* Исправлена проблема с finfo::file
* Исправлена проверка mime_type

= 1.3.2 =
* Исправлена проблема с функцией mime_content_type

= 1.3.1 =
* Исправлена проблема с отображением характеристик товаров
* Исправлена проблемы с потерей пользовательских изменений при очередном импорте
* Исправлены проблемы с повторяющимися категориями-вендорами
* Более информативные сообщения об ошибках при скачивании файла

= 1.3 =
* Исправлены проблемы при работе с ЧПУ
* Добавлены фильтры, позволяющие выбрать, какие именно товары из выгрузки окажутся в магазине
* В урлах применяется транслитерация
* Теги <param> из выгрузки включены в описание товара. В этих тегах содержится инфорамцяи по характерестикам товаров (вес, цвет, емкость аккумулятора и т. д)

= 1.2 =
* Уровень вложенности можно задавать в виджете.

= 1.1 =
* Возможность задавать уровень вложенности меню.
